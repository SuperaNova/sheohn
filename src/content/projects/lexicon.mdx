## Overview

Engineered a tree-walking custom language interpreter from scratch in Rust, utilizing an Abstract Syntax Tree (AST) architecture for robust parsing and execution.

## Web IDE Integration

Architected an interactive web playground using React and Astro, seamlessly integrating the core Rust engine via WebAssembly (Wasm) to deliver incredibly fast, real-time client-side code execution.
