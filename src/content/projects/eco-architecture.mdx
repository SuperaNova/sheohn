## Overview

The EcoArchitecture Core is a conceptual platform designed for high-end architectural firms that focus on sustainable building materials. The goal was to create a seamless e-commerce experience that feels less like a traditional store and more like an interactive gallery.

## The Challenge

Architectural materials are often sold through outdated B2B portals. The client needed a platform that:

1. Showcased the raw beauty of natural materials (wood, stone, recycled composites).
2. Allowed architects to easily calculate required quantities based on square footage.
3. Integrated seamlessly with their existing ERP system for real-time inventory.

## Architecture

We chose **Next.js** for its robust server-side rendering capabilities and excellent SEO performance, which was critical for the client's marketing strategy.

### Tech Stack Breakdown

- **Frontend:** Next.js (App Router), Tailwind CSS for styling, Framer Motion for subtle, premium animations.
- **Backend:** Node.js API connecting to a **PostgreSQL** database managed via Prisma ORM.
- **Infrastructure:** Deployed on Vercel for edge caching and fast global response times.

## Key Features

- **Dynamic Material Calculator:** Built a custom React hook that calculates material needs and price in real-time as the user inputs room dimensions.
- **Interactive Gallery:** Implemented a WebGL-powered material viewer allowing users to see textures under different lighting conditions.

## Conclusion

This project is currently in the late stages of development, with a planned launch in Q3. It represents a significant step forward in modernizing B2B material procurement.
