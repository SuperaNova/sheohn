## Overview

Crucible is an automated multi-agent pipeline architected using Google ADK to orchestrate state handoffs across vision and diffusion models. A data validation heuristic was engineered for strict quality control prior to inference.

## Optimization

Local model execution was optimized via float16 quantization to strictly bound memory footprints under 6GB VRAM. This enforced reproducible generation using fixed-seed API configurations and structured LLM reasoning.
